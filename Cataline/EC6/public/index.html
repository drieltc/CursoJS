Explicação Rápida

Basicamente eu vou codar meu JavaScript no arquivo main.js na versão mais recente do ES, e através de uma computaria muito louca esse código será transformado em ES5, disponível no arquivo bundle.js. Para que a computaria ocorra é necessário seguir o passo-a-passo abaixo:

Passo-a-Passo

  1. Abrir o WPS na pasta correta digitando o seguinte comando assim que abrir o terminal:

    cd C:\Users\Adriel\Documents\Cursos\CursoJS\Cataline\EC6

  2. Rodar o seguinte comando:

    .\node_modules\.bin\babel ./src/main.js -o ./public/bundle.js -w

    Alternativamente, pode-se rodar o seguinte comando, já que no nosso arquivo package.json a propriedade scripts tem como informações "dev" : "babel ./src/main.js -o ./public/bundle.js -w"

    npm run dev
